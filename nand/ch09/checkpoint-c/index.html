Checkpoint C

A quiz on Jack, including classes.

Class Creation

Create a class called Job with three fields:

  • An Array of integers called jobtimes that represents how long each job takes (in hours). It is possible for there to be job of time 0.
  • An integer numjobs that represents how many jobs there are currently in the array.
  • An integer maxjobs that represents the greatest number of jobs that can be in the array.

A constructor new takes in the int maximum number of of jobs.

A method void addJob(int hours) adds a new job. This means add the number to the jobtimes array and also incremenent the numjobs.

jobtimes == [10,15,6,0,0];  numjobs == 3;
jobtimes.addJob(13);
jobtimes == [10,15,6,13,0];  numjobs == 4;

If the maximum number of jobs has been reached and addJob is called, then the program should give error -3 (which causes it to terminate).

A method int getNumJobs() returns how many jobs there are currently.

A method int getJob(int k) returns the duration of job k, where k=0 is the first job.

A method void removeJob() removes the first job, job 0, from the list, moving each other job time to the previous spot in the list. (So job 1 with time 15 hours becomes job 0 with time 15 hours.)

A method int totalJobTime() to find the total amount of time all of the jobs together will take.
