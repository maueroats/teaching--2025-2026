02 Solutions

Solutions to the final exam for Nand to Tetris. (50 min)

Questions

  1. What is the difference between a function and a method in Jack?

    1. From a programming point of view?

    A function can only access static variables. A function is called on a class, not an object.

    A method can access field and static variables. A method is called on an object.

    1. At the virtual machine level?

    The method has an “invisible” first argument this pointing to the object on which it is being called.

     do Class.func(10)
 => push constant 10; call Class.func 1

 do obj.func(10)
 => push pointer 0; push constant 10; call Class.func 2

  2. Write a constructor new for the JT class that takes in two integers and assigns them to x and y respectively.

constructor JT new (int a, int b) {
    let x = a;
    let y = b;
    return this;
}

  1. The book mentions that the String class has a dispose() method as well as a new constructor.

    1. What is the purpose of the dispose method?

    The dispose method deallocates the memory that was allocated for the String when it was created.

    1. Why is it necessary? (What can happen if you ignore its existence?)

    Memory is not re-used if the dispose method is not called. If you have a String created inside a loop, the repeated allocations without deallocation can use up all available memory.

Write the rest of the Hack assembly code that results from a call. Include before and after the call.

     class Main {

         // create two JT variables, one and two.

 }

    Answer:

     // push local 0 (one)
 // push local 1 (two)
 // push ARG LCL THIS THAT

 // D = return address
 @RET.123
 D=A
 // push D
 @SP
 A=M
 M=D
 @SP
 M=M+1
 // adjust ARG = SP - 7
 @7
 D=A
 @SP
 D=M-D
 @ARG
 M=D
 // call JT.compareTo 2
 @JT.compareTo
 0;JMP
 // label RET.123
 (RET.123)
 // pop unused return value
 @SP
 M=M-1

  2. Write the parse tree for:

     class PQ {
     static int a;
     function void f() {
         var int a,b,c,d;
         let c = 10;
         return;
     }
     method int b(int c) {
         return a + c;
     }
 }

    Work:

'(class (identifier "PQ")
   (classVarDec (keyword "static") (keyword "int") (identifier "a"))
   (subroutineDec (keyword "void") (identifier "f") (parameterList)
                  ((varDec (keyword "int")
                           (identifier "a")
                           (identifier "b")
                           (identifier "c")
                           (identifier "d")))
                  ((letStatement (identifier c) (expression (term (integerConstant 10))))
                   (returnStatement)))
   (subroutineDec (keyword "int") (identifier "b") (parameterList ((keyword "int") (identifier "c")))
                  ((returnStatement (expression (term (identifier "a")) (op "+") (term (identifier "c")))))))
